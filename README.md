# Simple Web3 Wallet Checker

This is a very simple blockchain project for beginners.

This project helps you:
- Connect a crypto wallet (like MetaMask)
- Show wallet address
- Show wallet balance
- Show current blockchain network

This project is good for:
- Web3 beginners
- Blockchain learners
- Crypto developers who want practice

---

## What You Need

- A web browser
- MetaMask wallet installed
- Basic knowledge of HTML and JavaScript

---

## How To Run

1. Download or clone this repository
2. Open the folder
3. Open `index.html` in your browser
4. Click the "Connect Wallet" button
5. Allow MetaMask to connect

---

## Project Files

- `index.html` = Main web page  
- `app.js` = Blockchain logic  
- `config.js` = Network settings  
- `wallet.js` = Wallet connection code  
- `balance.js` = Balance checker  
- `network.js` = Network detector  
- `utils.js` = Helper functions  
- `style.css` = Simple styles  
- `logger.js` = Console logs  
- `README.md` = Project guide
